# Happy Holi 🌈✨

A beautiful, interactive Happy Holi celebration webpage with colorful animations and effects.

## Features

✨ **Animated gradients** - Smooth color transitions across the background  
🎨 **Color splash effects** - Click the button to create colorful popping bubbles  
🎵 **Background music** - Festive music plays automatically  
💫 **Smooth animations** - Bouncing and floating text effects  
📱 **Responsive design** - Works on all devices  

## How to Use

### Open locally:
1. Download the `index.html` file or clone this repository
2. Double-click `index.html` to open in your browser *or* serve it over HTTP for best results (some browsers restrict audio/files when opened directly).

   To start a simple local server with Python, run:
   ```bash
   # Python 3
   python3 -m http.server 8000
   # then visit http://localhost:8000/index.html in your browser
   ```
3. Click the "Tap for Colors 🎨" button and move your mouse to enjoy all the splashes, confetti, and water‑gun animation.

### Customize:
Edit the `index.html` file to change:
- **Name**: Replace "For Someone Special 🌸" with a name
- **Message**: Update the greeting text 
- **Colors**: Modify the color scheme in the CSS gradient

### Host online (free options):
- **GitHub Pages**: Push to your repository and enable GitHub Pages in Settings
- **Netlify**: Drag and drop `index.html` on [netlify.com](https://netlify.com)
- **Vercel**: Deploy on [vercel.com](https://vercel.com)

## Technologies Used
